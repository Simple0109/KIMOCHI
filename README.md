# [Matoi]

## サービス概要
Matoiはユーザーが所有しているもしくは、欲しいと思っているドメスティックブランドの洋服の画像を情報（ブランド名、ポイント）を追記して投稿できます。

## 想定されるユーザー層
20代後半〜40代の男女
ファストファッションブランドではなく、愛着を持って長く着れるドメスティックファッションブランドの洋服を探している人。

## サービスコンセプト
ファストファッションの流行によって、洋服が大量生産、大量廃棄されています。
ファストファッションを否定するわけではありませんが、もっと愛着を持って長く着れる国産の洋服に目を向けてもいいのではないかと考えています。
しかしそのような洋服の情報はファストファッションと比べると情報量が少なく、「着用してみたいが、どんなものがいいかわからない。なにがいいかわからない。」といった悩みを抱えている人もいるかと思います。
このアプリは登録したユーザーが自身が購入した服、もしくは着用してみたい服をその服の特徴と一緒に投稿することができ、気軽にどのような服なのか、ブランドなのかを知ることができます。
またユーザーが購入したor取り扱いのあるセレクトショップのECサイト、もしくはブログを表示することで実際に商品を手にとってみることへのハードルを下げることが可能です。
Matoiではこれまでドメスティックブランドに興味がなかった人も、既にドメスティックブランドを着用している人も、お互い情報を共有できるアプリになっています。

## 実装を予定している機能
### MVP
* 会員登録
* ログイン
* 投稿一覧
* 投稿詳細
* コメント機能
* フォロー機能

### その後の機能
* マルチ検索
* タグ検索
* レコメンド機能
* 実店舗住所地図表示機能
